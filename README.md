# Sistema de Produtos em Java

## Sobre o projeto

Este projeto consiste em um sistema simples de gerenciamento de produtos desenvolvido em Java, utilizando conceitos fundamentais da programação, como estruturas de repetição, arrays e programação orientada a objetos (POO).

O sistema permite cadastrar, listar e buscar produtos por nome, simulando um pequeno sistema de controle de estoque.

---

## Funcionalidades

* Cadastro de produtos (nome e preço)
* Listagem de todos os produtos cadastrados
* Busca de produtos por nome (ignorando maiúsculas/minúsculas)
* Controle de limite (máximo de 5 produtos)
* Menu interativo em loop

---

## Conceitos aplicados

* Estruturas de repetição (`while`, `for`)
* Estruturas condicionais (`if`, `else`)
* Arrays de objetos
* Programação Orientada a Objetos (POO)
* Entrada de dados com `Scanner`

---

## Exemplo de uso

```text
1 - Cadastrar produto
2 - Listar produtos
3 - Buscar produtos
4 - Sair

Escolha: 1

Digite o nome do produto: Arroz
Digite o preço do produto: R$20.50

Produto cadastrado com sucesso!
```
